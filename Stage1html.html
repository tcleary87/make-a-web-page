Lesson 1: The Basics of the Web and HTML

The World Wide Web

The web, invented in the early 90s, is a large network of tons of computers that are all able to communicate with one another. Computers send requests to each other using HTTP(Hypertext Transfer Protocol).

HTML

HTML (Hypertext Markup Language) is the language used to form the structure of most pages on the internet. HTML consists of text content and the markup which affects its layout.

Tags and Elements

HTML documents use tags like <b> and <p> to define the layout of the content. These tags, in combination with their content, makeup HTML elements. Most pages are made up of many elements. Most tages have content, but some tags like <img> do not contain content and are considered void.

Silly Computers. Intuition is for Humans.

Computers, while powerful, are actually very simplistic and lack the intuition of humans. Computers only posess the ability to follow all orders literally and can therefore be very unforgiving of mistakes made in programming.

Inline vs. Block Elements

While inline elements, such as an image, do not affect the spacing, block elements place their content in an invisible box that can be modified.

Lesson 2: Creating a Structured Document with HTML

The DOM

The DOM (Document Object Model) is the cross platform, cross language, standard convention that is responsible for the tree like structure of the page.

Boxes Inside of Boxes

The treelike structue exists becuase the HTML divides the page into different boxes, and the boxes can be nested inside of one another. This allows for logical structure that can later be styled by CSS.

Using the Right Tools

Because HTML is so specific, the syntax must be exactly correct for the results to be as intended. It is also important that the tree like structure of the boxes on the page be reflected in the indentation of the elements in the HTML. Luckily, there are special text editors, such as Sublime Text, that can make this easier for the programmers using them.

Lesson 3: Adding CSS Style to HTML Structure

Cascading Style Sheets

Cascading Style Sheets, or CSS, is code written to control the style of HTML elements. With a CSS declaration, the programmer is able to select HTML elements and assign properties various values.

The Box Model

Becuase HTML elements are held in boxes, it is important to understand the box model when using CSS to set your page layout. The content is surrouded by a padding which is surrounded by the border which is surrounded by a margin.

Don't Repeat Yourself!

When desiging a webpage, you want to avoid as much repetition as possible in your CSS. If you need to make a change to something you want to be able to do it once and have it affect as much as possible and not have to worry about changing the style for each individual element. By organizing your HTML into the proper elements with well chosen classes you can save yourself a lot of trouble.
